# Roblox Manager

Multi-account Roblox manager with built-in auto-rejoiner, item-detection, and account rotation.

## Download

Latest release: <https://github.com/YungTedo/RobloxManager-Downloads/releases/latest>

Direct exe link: <https://github.com/YungTedo/RobloxManager-Downloads/releases/latest/download/Roblox%20Manager.exe>

## How to use

1. Download `Roblox Manager.exe` from the latest release
2. Run it (Windows 10/11 only)
3. Account Manager features (add accounts, manage games, browser login, RAM import) work without a license
4. To use the auto-rejoiner, click the **Roblox Alive** tab and enter your license key
